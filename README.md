# springcloud微服务商城

#### 介绍

练习的综合项目 主要熟悉微服务框架SpringCloud与练习其他技术

#### 使用技术

* 后端：SpringBoot + SpringCloud + TkMybatis + FastDfs + Thymeleaf + Redis + Mongodb +Elstaticsearch + Rabbitmq + Jwt

* 前端：Vue + axios + live-server + qrcode
